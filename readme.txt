Production-ready 3D showcase plugin with shortcode support, admin settings, and scalable architecture.

== Description ==
Simple 3D Showcase provides a modular architecture for rendering GLB/GLTF models with model-viewer. It includes:

* Backward-compatible shortcode: [simple_3d_showcase]
* Recommended shortcode: [simple_3d_showcase id="123"]
* 3D Showcase custom post type
* Dashboard settings page with global defaults
* REST API foundation endpoint
* Extensible architecture for future features (hotspots, explode view, presets, kiosk flows)
